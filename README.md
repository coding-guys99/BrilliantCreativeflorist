# Florist Gallery Site (3 languages)

High-end, fast, mobile-friendly gallery website for a florist.
- Languages: 简体 / 繁體 / English
- Desktop: 5 images per row
- Mobile: responsive 2–3 columns
- 1:1 thumbnails + lightbox viewer (swipe on mobile)
- WhatsApp CTA + WhatsApp QR image

## Folder
- `assets/gallery/` put your images here
- `data/gallery.json` update your filenames & text
- `assets/ui/whatsapp-qr.png` replace with your WhatsApp QR image
- `data/site.json` update address/hours/WhatsApp link

## Deploy with GitHub Pages
1. Push to GitHub repo
2. Settings → Pages → Build and deployment → Deploy from a branch
3. Select `main` branch and `/ (root)`
4. Your site URL will be: `https://USERNAME.github.io/REPO/`

## Notes
- For best look: use images 1500px+ (jpg/webp).
- Keep filenames consistent with `gallery.json`.
